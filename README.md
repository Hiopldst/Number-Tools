# Number-Tools

## Description

A simple console Python project with a menu that allows the user to choose different operations.

The program uses functions, loops, conditions, and user input.

## Features

* Multiplication table generator
* Factorial calculator
* Square numbers generator
* Cube numbers generator
* Interactive menu
* Exit option

## Technologies

* Python 3

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the Python file:

```bash
python main.py
```

4. Choose an option from the menu.
